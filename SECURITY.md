# Security

Themis is a demonstration project for read-only infrastructure change review. It should not receive real secrets, customer data or confidential infrastructure details.

If you find an underlying vulnerability in this repository, report it through GitHub's private vulnerability reporting via https://github.com/steadytao/themis-agent/security/advisories/new

Do not send exploit payloads, credentials, private keys, customer data or embargo-sensitive material in public issues.
