# Community clinic admin portal exposure

Service: appointment admin portal
Environment: production

A small community clinic wants to make its internal appointment-admin portal reachable through a public Azure Application Gateway so remote staff can access it. TLS will terminate at the edge. The network security group will temporarily allow 0.0.0.0/0 while staff test access.

The portal handles staff administration and patient appointment details. Authentication will be added later after the routing path is confirmed.
